# الدين والأخلاق — دليل المذاكرة • Din Kültürü ve Ahlak Bilgisi Rehberi

صفحة مذاكرة تفاعلية ثنائية اللغة (تركي ↔ عربي) لمساعدة الطالب على فهم وحفظ محتوى مادة **الدين والأخلاق** للصف السادس والاستعداد للامتحان.

An interactive bilingual (Turkish ↔ Arabic) study page for 6th-grade **Religion & Ethics** (Din Kültürü ve Ahlak Bilgisi) exam revision.

## 🚀 الاستخدام

افتح **[`din-ahlak-rehberi.html`](./din-ahlak-rehberi.html)** بالنقر المزدوج — تعمل دون إنترنت، ولا تحتاج أي تثبيت.

> 🌐 نسخة الويب: <https://yahyaabohashemstu.github.io/dinfinal/>

## 📐 التصميم

- **التركية على اليسار، والعربية على اليمين، متقابلتان تمامًا** — كل جملة تركية تواجه ترجمتها العربية بدقة.
- الكلمات المهمة (أجوبة الامتحان) مُظلّلة بالذهبي.

## ✨ المزايا التفاعلية

- 🔑 **بطاقات المفاهيم** — اضغط البطاقة لتنقلب وترى المعنى.
- 🔊 **النطق التركي** — استمع لكل كلمة بصوت تركي صحيح (`speechSynthesis`).
- 🧭 **خط زمني تفاعلي** لطفولة النبي ﷺ (مَن رعاه في كل مرحلة).
- 📝 **اختبارات قصيرة** اختيار من متعدد بتصحيح فوري وشرح.
- 🎯 **لعبة المطابقة** بين المفهوم التركي ومعناه العربي.
- ✓ **تتبّع التقدّم** + 🌙 **الوضع الليلي** — محفوظ تلقائيًا على الجهاز.
